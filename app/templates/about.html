ASTU Digital Lost & Found System

About the Project

The ASTU Digital Lost & Found System is a web-based platform designed to help students, faculty, and staff at ASTU (Adama Science and Technology University) to report, search, and recover lost personal belongings on campus.

Features

  • Report lost or found items with images
  • Search and filter items by category, location, and date
  • Submit claims for items you believe are yours
  • Admin dashboard for managing claims and items
  • User authentication and profile management

Technology Stack

  • Backend: Python Flask
  • Database: SQLite with SQLAlchemy ORM
  • Frontend: HTML, Bootstrap 5, Jinja2
  • Authentication: Flask-Login

Security Features

  • Password hashing with Werkzeug
  • CSRF protection with Flask-WTF
  • Role-based access control (RBAC)
  • Secure file upload handling
  • Input validation
